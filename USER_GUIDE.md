
# WriteCraft User Guide

## Getting Started

### Creating Your Account

1. Visit the WriteCraft homepage
2. Click "Sign In" in the top-right corner
3. Choose your authentication method (Google, GitHub, etc.)
4. Complete the authentication flow

### Understanding Notebooks

**Notebooks** are containers for organizing your worldbuilding content. Think of them as separate universes or story settings.

**Best Practices:**
- Create one notebook per story/series
- Use descriptive names (e.g., "The Crimson Saga", "Cyberpunk 2184")
- Keep related content in the same notebook

### Understanding Projects

**Projects** are where you write your actual manuscripts. Projects are separate from notebooks.

**Key Features:**
- Hierarchical structure (folders and pages/sections)
- Rich text editor with formatting
- Auto-save
- Word count tracking
- AI-powered writing assistance

## Core Features

### 1. Notebooks & Worldbuilding

#### Creating a Notebook

1. Click the notebook switcher in the header
2. Select "Create New Notebook"
3. Enter a name and optional description
4. Click "Create"

#### Adding Content to a Notebook

1. Select your active notebook from the switcher
2. Navigate to a content type (Characters, Locations, etc.)
3. Click "Create New [Type]"
4. Fill in the form fields
5. Click "Save"

**Available Content Types:**

- **Characters**: People, creatures, and beings in your world
- **Locations**: Places, buildings, and geographic features
- **Items & Weapons**: Objects, artifacts, and equipment
- **Organizations**: Groups, factions, and institutions
- **Cultures & Species**: Peoples, races, and societies
- **Magic & Technology**: Spells, tech, natural laws
- **History**: Events, timelines, myths, legends
- **And 40+ more categories**

### 2. Writing Projects

#### Creating a Project

1. Go to the Projects page
2. Click "New Project"
3. Enter a title
4. Start writing!

#### Project Structure

Projects use a **hierarchical outline**:

- **Folders**: Organize sections (e.g., "Act 1", "Character Arcs")
- **Pages/Sections**: Individual writing sections (e.g., "Chapter 1")

**To add structure:**

1. Click the outline icon in the project editor
2. Use the "+" button to add folders or pages
3. Drag and drop to reorganize
4. Click any section to start writing

#### Writing & Editing

The editor includes:

- **Rich text formatting**: Bold, italic, headings, lists, etc.
- **Auto-save**: Changes save automatically every few seconds
- **Word count**: Live word count in the status bar
- **AI assistance**: Select text and use the AI menu for suggestions

### 3. AI-Powered Features

#### AI Text Editing

1. Select any text in the editor
2. Click the AI sparkle icon that appears
3. Choose an action:
   - **Improve**: Enhance clarity and style
   - **Shorten**: Make more concise
   - **Expand**: Add detail and depth
   - **Fix Grammar**: Correct errors

#### Writing Assistant

The Writing Assistant is your AI writing coach:

1. Click the Writing Assistant icon in the toolbar
2. Choose a mode:
   - **Chat**: Ask questions, brainstorm ideas
   - **Analyze**: Get feedback on selected text
   - **Proofread**: Detailed grammar and style review
   - **Questions**: Generate questions about your text

**Example Uses:**
- "How can I improve the pacing in this scene?"
- "What's a better way to describe this character's motivation?"
- "Analyze this dialogue for authenticity"

#### Content Generators

Generate worldbuilding content with AI:

1. Navigate to Generators page
2. Choose a generator (Characters, Locations, etc.)
3. Fill in prompts and parameters
4. Click "Generate"
5. Save results to your notebook

### 4. Organizing Your Work

#### Saving Generated Content

When you generate content (characters, locations, etc.):

1. Review the generated content
2. Click "Save to Notebook"
3. Select which notebook to save to
4. Edit and customize as needed

#### Saved Items Library

Access all your saved content:

1. Go to "My Content" or use the notebook switcher
2. Filter by content type
3. Click any item to view or edit
4. Use search to find specific items

#### Notes

Create quick notes for ideas:

1. Click the Quick Note panel
2. Type your notes
3. Auto-saves as you type

Create structured notes:

1. Go to Projects or Guides
2. Click the notes icon
3. Create notes attached to specific documents

### 5. Writing Guides

Access educational resources:

1. Navigate to the Guides page
2. Browse by category or difficulty
3. Click a guide to read
4. Search for specific topics

**Categories:**
- Plot Structure
- Character Development
- Dialogue
- Description
- Genre-specific techniques

## Tips & Best Practices

### Worldbuilding Workflow

1. Start with core concepts (setting, magic system, major factions)
2. Create main characters and locations
3. Build out supporting elements as needed
4. Link related content using the mention system (`@` in editor)

### Writing Workflow

1. Outline your project structure first (acts, chapters, scenes)
2. Write in drafts - don't worry about perfection
3. Use AI assistance for revision, not initial drafting
4. Save milestones by exporting periodically

### Organization Tips

- **Use descriptive names**: "Tavern Scene - Chapter 3" not "Scene 1"
- **Tag consistently**: Use the same genre/tags across related content
- **Review regularly**: Periodically review and update old content
- **Export backups**: Download your work periodically

## Keyboard Shortcuts

### Editor

- `Ctrl+B` - Bold
- `Ctrl+I` - Italic
- `Ctrl+K` - Insert link
- `Ctrl+/` - Toggle comment
- `Ctrl+Z` - Undo
- `Ctrl+Shift+Z` - Redo
- `Ctrl+S` - Save (auto-saves, but forces immediate save)

### Navigation

- `Ctrl+P` - Quick open (search files)
- `Ctrl+Shift+F` - Global search
- `Ctrl+,` - Settings

## Troubleshooting

### Content Not Saving

- Check your internet connection
- Look for error messages in red
- Try refreshing the page
- Contact support if issue persists

### AI Features Not Working

- Verify you're logged in
- Check if you have AI features enabled
- Some features require active subscription

### Can't Find Content

- Verify you're in the correct notebook
- Use the search function
- Check content type filters
- Ensure content was saved (not just generated)

## Account Management

### Profile Settings

1. Click your avatar in the top-right
2. Select "Account Settings"
3. Update your profile information
4. Manage display preferences

### Data Export

Export your work:

1. Open a project or guide
2. Use the export function
3. Choose format (Markdown, PDF, etc.)
4. Download to your device

### Account Deletion

To delete your account:

1. Go to Account Settings
2. Scroll to "Danger Zone"
3. Click "Delete Account"
4. Confirm deletion

**Warning**: This permanently deletes all your data and cannot be undone.
